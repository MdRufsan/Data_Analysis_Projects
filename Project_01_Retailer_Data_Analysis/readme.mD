# 📊 Beverage Retail Sales Analysis

**Comprehensive Sales Performance Analysis · January 2022 – December 2022**

---

## 📋 Project Overview

This project presents a comprehensive sales performance analysis of beverage products across four major retail chains: **Costco**, **Target**, **Walgreens**, and **Walmart**. The analysis encompasses sales data from **January 2022** through **December 2022**, providing insights into revenue patterns, product performance, regional market dynamics, and operational metrics.

## 📐 Data Schema

| Field | Description | Data Type |
|-------|-------------|-----------|
| Retailer | Retail chain name | String |
| Contact | Account representative | String |
| Retailer ID | Unique retailer identifier | Integer |
| Order Date | Date of order placement | Date |
| Payment Date | Date of payment receipt | Date |
| Region--State | Geographic location | String |
| Beverage Brand | Product name | String |
| Price per Unit | Unit price in USD | Float |
| Units Sold | Quantity purchased | Integer |
| Revenue | Total revenue (calculated) | Float |

## 🔑 Key Findings

### 💰 Revenue Performance
- **Total Revenue:** $1,213,017.50
- **Top Performer:** Walmart ($391,767.50)
- **Second:** Costco ($331,750)
- **Third:** Walgreens ($250,650)
- **Fourth:** Target ($238,850)

### 🥤 Product Performance
- **Best-Selling:** Coca-Cola (461,900 units)
- **Second:** Diet Coke (413,750 units)
- **Third:** Dasani Water (392,750 units)
- **Highest Revenue:** Dasani Water ($233,775)
- **Highest Unit Price:** Dasani Water ($0.80 max)

### 📍 Regional Insights
- **West Region:** 1,037,250 units (44.9%)
- **Northeast Region:** 706,600 units (30.6%)
- **South Region:** 566,000 units (24.5%)
- **Total Units Sold:** 2,309,850

### 💰 Price Analysis
- **Price Range:** $0.25 – $0.80
- **Highest Avg Price:** Dasani Water ($0.59)
- **Lowest Avg Price:** Coca-Cola ($0.48)

## ⚙️ Technical Implementation

- **Platform:** Microsoft Excel
- **Data Processing:** Pivot tables, formulas, and data validation
- **Dashboard Tools:** GETPIVOTDATA functions for dynamic data retrieval
- **Data Volume:** 288 transaction records across 4 retailers, 6 products, and 12 months
